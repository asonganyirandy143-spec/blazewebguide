GitHub Pages Deployment

Step 1: Create Repository

Create a new repository on GitHub, name it, and initialize with README.

Step 2: Push Local Files

Use Git to push your local website files to the repository’s main branch.

Step 3: Enable GitHub Pages

Go to repository settings → Pages → Select branch → Save.

Step 4: Add Custom Domain

If using www.blazewebguide.gleeze.com, add CNAME to your repository settings and configure DNS properly.
